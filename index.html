📖 Command Guide

วิธีใช้คำสั่ง Ariea

รวมคำสั่งทั้งหมด 13 คำสั่งของ Ariea พร้อมคำอธิบายว่าใช้ทำอะไร และใครใช้ได้บ้าง

💡 ทุกคำสั่งของ Ariea เป็น Slash Command — แค่พิมพ์ / ในช่องแชท แล้วเลือกชื่อคำสั่งจากรายการที่ Discord แสดงขึ้นมา ไม่ต้องจำ prefix ใดๆ ทั้งสิ้น
ใช้ได้ทุกคน
ต้องเป็นแอดมิน
เจ้าของบอทเท่านั้น
CMD 01
/
/help

แสดงรายการคำสั่งทั้งหมดของ Ariea พร้อมคำอธิบายแบบย่อ ใช้เมื่อจำคำสั่งไม่ได้หรืออยากรู้ว่าบอทใช้ทำอะไรได้บ้าง

ใช้ได้ทุกคน
CMD 02
/
/invite

รับลิงก์เชิญ Ariea ไปใช้งานในเซิร์ฟเวอร์อื่นได้ทันที ไม่ต้องออกไปหาลิงก์จากเว็บ

ใช้ได้ทุกคน
CMD 03
/
/ping

เช็คความหน่วง (latency) ของบอทในหน่วยมิลลิวินาที ใช้ดูว่าตอนนี้บอทตอบสนองไวแค่ไหน

ใช้ได้ทุกคน
CMD 04
/
/tiktok <ลิงก์>

วางลิงก์คลิป TikTok แล้วบอทจะดาวน์โหลดวิดีโอแบบไม่มีลายน้ำส่งเข้าแชทให้ทันที

ใช้ได้ทุกคน
CMD 05
/
/dmwatch <on/off>

เปิด-ปิดระบบดักข้อความ DM เมื่อมีคนทักหาบอทโดยตรง Ariea จะส่งข้อความนั้นแจ้งเตือนเข้าห้องที่กำหนดในเซิร์ฟเวอร์

ต้องเป็นแอดมิน
CMD 06
/
/vc247 <join/leave>

สั่งให้ Ariea เข้าไปอยู่ในห้องเสียงที่เลือกและออนไลน์ต่อเนื่อง 24 ชั่วโมง หรือสั่งให้ออกจากห้องเมื่อไม่ต้องการแล้ว

ต้องเป็นแอดมิน
CMD 07
/
/blacklist add <user>

เพิ่มผู้ใช้ที่ระบุเข้าบัญชีดำของเซิร์ฟเวอร์ ทำให้คนนั้นใช้คำสั่งของ Ariea ในเซิร์ฟเวอร์นี้ไม่ได้อีก

ต้องเป็นแอดมิน
CMD 08
/
/blacklist remove <user>

ปลดผู้ใช้ที่ระบุออกจากบัญชีดำ ทำให้กลับมาใช้คำสั่งของ Ariea ได้ตามปกติ

ต้องเป็นแอดมิน
CMD 09
/
/blacklist list

แสดงรายชื่อผู้ใช้ทั้งหมดที่อยู่ในบัญชีดำของเซิร์ฟเวอร์นี้ ใช้ตรวจสอบว่าใครถูกแบนไว้บ้าง

ต้องเป็นแอดมิน
CMD 10
/
/serverinfo

แสดงข้อมูลของเซิร์ฟเวอร์ เช่น จำนวนสมาชิกทั้งหมด วันที่สร้างเซิร์ฟเวอร์ และจำนวนห้องต่างๆ

ใช้ได้ทุกคน
CMD 11
/
/userinfo <@user>

แสดงข้อมูลของผู้ใช้ที่ระบุ เช่น วันที่เข้าร่วมเซิร์ฟเวอร์ วันที่สร้างบัญชี และ role ที่มี

ใช้ได้ทุกคน
CMD 12
/
/avatar <@user>

ดึงรูปโปรไฟล์ของผู้ใช้ที่ระบุมาแสดงแบบขนาดเต็ม ไม่ระบุ user จะเป็นการดึงรูปโปรไฟล์ของตัวเอง

ใช้ได้ทุกคน
CMD 13
/
/stats

แสดงสถิติการทำงานของ Ariea เช่น เวลาที่ออนไลน์ต่อเนื่อง (uptime) และจำนวนเซิร์ฟเวอร์ที่กำลังใช้งานบอทอยู่
